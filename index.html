🕹 원격 명령 Actions 직접 실행

🔑 GitHub 토큰 설정 (최초 1회)

Fine-grained 토큰 필요: Settings → Developer settings → Fine-grained tokens → 이 저장소만 선택 → Permissions에서 Actions: Read and write, Contents: Read-only(현황판 조회용), Secrets: Read and write(텔레그램 설정용). 토큰은 이 브라우저에만 저장되며 저장소에 올라가지 않습니다.

📱 텔레그램 설정 (봇 토큰 / chat_id 변경)

입력한 값은 GitHub Secrets에 암호화되어 저장됩니다. 먼저 아래 "🔑 GitHub 토큰 설정"에 Secrets: Read and write 권한이 포함된 토큰이 저장되어 있어야 합니다.

자동 감지: 위에 봇 토큰 입력 → 텔레그램에서 봇에게 아무 메시지나 전송 → 버튼 클릭
