# 무한매수법 정리 및 보완점 대비 전략

> 라오어의 무한매수법을 온라인 자료 기준으로 정리한 문서입니다.
> 정확한 원문 규칙은 책과 [무한매수법 네이버 카페](https://cafe.naver.com/infinitebuying)를
> 반드시 직접 확인하세요. (라오어님 공지에 따라 시트/규칙의 무단 재배포가 제한되어
> 있으므로, 이 저장소는 **개인 학습·기록용**으로만 사용하세요.)

## 1. 핵심 개념

- 대상: TQQQ, SOXL 같은 3배 레버리지 ETF
- 원금을 여러 회차로 분할해 **매일 기계적으로** LOC 주문을 걸고,
  평단가 대비 일정 수익률에 도달하면 전량 매도 후 사이클을 재시작
- 핵심 지표 **T값** = 누적매수액 ÷ 1회 매수금 (소수점 둘째 자리에서 올림)
  → "원금을 몇 회차까지 소진했는가"

### LOC(Limit On Close) 주문
장 마감 종가가 내 지정가보다 **유리할 때만** 종가로 체결되는 주문.
- LOC 매수 $100: 종가 ≤ $100이면 종가에 체결
- LOC 매도 $100: 종가 ≥ $100이면 종가에 체결
- 한국투자증권, 토스증권 모두 미국주식 LOC 주문을 지원합니다.

## 2. 버전별 규칙 요약

### v2.2 (2022.12, 안정형 — 이 저장소의 기본값)

| 구분 | 규칙 |
|---|---|
| 분할 | 원금 40분할 (1회 매수금 = 원금/40) |
| 전반전 매수 (T<20) | 절반: 평단가 LOC / 절반: 평단×(1+(10−T/2)%) LOC |
| 후반전 매수 (T≥20) | 전액: 평단×(1+(10−T/2)%) LOC |
| 매도 (매일) | 1/4: 평단×(1+(10−T/2)%) LOC + 3/4: 평단+10% 지정가 |
| 원금 소진 시 | 쿼터 매도로 시드 재확보 후 지속 |

(10−T/2)%는 T가 커질수록 낮아져서, 오래 물릴수록 목표 수익률을 낮춰
탈출을 앞당기는 구조입니다. T>20부터는 음수가 되어 사실상 부분 손절이
자동으로 일어납니다.

### v3.0 (2024, 공격형)

| 구분 | 규칙 |
|---|---|
| 분할 | 원금 20분할, 수익금은 40분할해서 다음 1회 매수금에 즉시 복리 반영 |
| 매수 기준 | (15−1.5T)% |
| 매도 | 1/4: (15−1.5T)% LOC + 3/4: TQQQ +15% / SOXL +20% 지정가 |
| 쿼터모드 (19<T<20) | 1/4 MOC 매도, 매수 중단 |

⚠️ v3.0은 20분할이라 하락장에서 원금 소진이 훨씬 빠릅니다. 공개 블로그
요약 기반으로 구현했으므로 세부 규칙(쿼터모드 예외 등)은 카페 원문과
대조 후 사용하세요.

## 3. 알려진 약점과 대비책 (보완 전략)

### 약점 ① 3배 레버리지의 변동성 잠식 (Volatility Drag)
지수가 제자리여도 오르내림이 반복되면 3배 ETF는 녹습니다.
- **대비**: 무한매수법 시드는 **전체 금융자산의 10~20% 이내**로 제한.
  나머지는 지수 ETF·현금 등 안전자산으로 유지.

### 약점 ② 장기 하락장에서 원금 소진
2022년형 하락장에서는 40분할로도 20회차를 훌쩍 넘긴 사례가 많고,
백테스트 MDD는 −47% 수준입니다.
- **대비 1**: v2.2 유지 (v3.0의 20분할은 소진이 2배 빠름)
- **대비 2**: **예비 시드**를 별도로 보유 (시드의 30~50%). 원금 소진 시
  낮아진 평단에서 사이클을 잇는 데 사용.
- **대비 3**: TQQQ/SOXL **시드를 분리**해서 운영 (이 저장소 기본 구조).
  반도체와 나스닥의 하락 시점이 어긋나면 서로 완충이 됩니다.

### 약점 ③ 규칙 이탈 (실패 원인 1위)
폭락하면 무섭고 폭등하면 욕심나서 임의로 매매를 바꾸는 것이 최대 리스크.
- **대비**: 이 저장소의 자동화가 그 역할입니다. 매일 아침 텔레그램으로
  받은 주문을 **그대로만** 입력하고, 판단은 하지 않기. 주문 입력에
  걸리는 시간은 하루 3분입니다.

### 약점 ④ 체결 오차·기록 누락
LOC 체결 추정과 실제 체결이 다를 수 있습니다 (호가 단위, 부분 체결,
주문 누락한 날 등).
- **대비**: **매주 토요일** 증권사 잔고(보유수량·평단)와 웹앱 표시를
  대조. 다르면 `data/state.json`의 `shares`, `avg_price`,
  `total_bought`를 실제 값으로 수정 후 커밋.

### 약점 ⑤ 상장폐지/괴리 리스크
3배 ETF는 폭락 시 청산(상폐) 가능성이 이론적으로 존재합니다.
- **대비**: 하루 −33% (기초지수 −11%)를 넘는 폭락이 나오면 규칙과
  무관하게 상황을 점검. 시드 분산(TQQQ+SOXL)도 부분 완충이 됩니다.

### 약점 ⑥ 세금·환율
- 해외주식 양도소득세: 연 250만원 공제 후 22%. 무한매수법은 회전이
  많아 **수익이 나면 세금이 반드시 발생**합니다. 연말에 손익통산 확인.
- 환율: 시드를 달러로 환전해두고 원화 환산 손익에 일희일비하지 않기.

### 약점 ⑦ 자동화 시스템 자체의 실패
GitHub Actions 미실행, 야후 시세 오류 등.
- **대비**: 봇이 실패해도 웹앱의 **수동 계산기**로 즉시 같은 주문을
  계산할 수 있습니다. 텔레그램 메시지가 아침에 안 오면 Actions 탭에서
  수동 실행(workflow_dispatch) 하세요.

## 4. 시작 전 체크리스트

- [ ] 시드가 전체 자산의 20% 이하인가?
- [ ] 예비 시드(시드의 30% 이상)를 따로 두었는가?
- [ ] 이 돈이 반토막 나도 생활에 지장이 없는가?
- [ ] 증권사에서 레버리지 ETF 사전교육 이수 및 거래 등록을 했는가?
- [ ] 최소 1~2사이클은 소액으로 테스트할 것인가?
